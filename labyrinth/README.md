### Environment

* Http Server
* Node.js
* Browser `Chrome、Safari、Firefox、Opera、Edge`

### Publish

**Install Dependence**

  * [Homebrew](http://brew.sh/)
  * [Node.js](https://nodejs.org/) `brew install node`
  * [gulp](http://gulpjs.com/)  `npm install -g gulp`
  * `npm install` // node_modules

**Build**

  * `gulp`
  * `gulp dev` // local develop
